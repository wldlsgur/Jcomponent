# Project Name

JiPonent는 React에서 컴포넌트, 커스텀 훅 기능을 제공합니다. <br>
컴포넌트는 스타일만 커스텀하여 사용하시면 됩니다.

## Installation

```
npm install jiponent
```

## Example

```jsx
import { ComponentName, HookName } from 'jiponent';

function App() {
  const customHookValue = HookName();

  return <ComponentName />;
}
```
